*****************  database deleting process
*** Active Connections: PostgreSQL prevents dropping a database if there are active connections to it. This is a safety measure to prevent data loss or corruption. To drop the database, all connections to it must first be terminated. To identify active connections, query pg_stat_activity:

SELECT pid, usename, datname FROM pg_stat_activity WHERE datname = 'my_datawarehouse';

*** To terminate connections, use pg_terminate_backend():
SELECT pg_terminate_backend(pid) FROM pg_stat_activity WHERE datname = 'my_datawarehouse';
